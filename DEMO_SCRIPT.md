# Demo Script (April 4 Live)

## 3-minute run of show

1. Open the app and show onboarding welcome.
   - Say: "Your money stays on your device. No cloud sync, no bank login."
2. Land on Dashboard.
   - Point out Safe to Spend, donut chart, and recent transactions.
3. Go to Settings.
   - Enter primary bank (example: "America First").
   - Open Ignore apps and toggle one app (example: Venmo).
4. Trigger an incoming notification from test tooling.
   - Return to Settings and tap Process Now.
   - Narrate processing status text while parsing runs.
5. Show the created transaction in Dashboard/Transactions.
   - Highlight AI sparkle and confidence indicator.
6. Edit the transaction category in the bottom sheet.
   - Explain this saves a custom rule to improve future categorization.
7. Go to Budgets.
   - Show progress bar changing color near/over limits.
8. Close with privacy statement.
   - "All parsing and storage are local, encrypted, and controlled by the user."

## Dry-run checklist

- Device has listener permission enabled.
- POST_NOTIFICATIONS permission granted.
- At least one pending raw notification exists before demo.
- Low budget cap set for one category to show warning state.
- Screen brightness and font scale checked for camera capture.
